Générateur de Clé API Chiffrée

Information: Cette page vous permet de créer un fichier api.key chiffré qui contient votre clé API OpenAI. Utilisez un mot de passe fort et mémorisez-le, car vous en aurez besoin pour déverrouiller la clé dans demo_gpt.html.
Succès! Le fichier api.key a été généré et téléchargé. Placez-le dans le dossier docs/ et utilisez-le dans demo_gpt.html avec le mot de passe que vous avez défini.

Instructions:

  1. Entrez votre clé API OpenAI (commence par "sk-")
  2. Choisissez un mot de passe fort et mémorisez-le
  3. Confirmez le mot de passe
  4. Cliquez sur "Générer et Télécharger api.key"
  5. Placez le fichier téléchargé (api.key) dans le dossier docs/
  6. Dans demo_gpt.html, utilisez le mot de passe pour déverrouiller la clé
